Linux其他命令

软链接

ln命令在系统中创建软链接（标记为l），可以将文件、文件夹链接到其他位置。

  • 参数1，表示被连链接的原文件或文件夹
  • 参数2，表示要链接去的文件或文件夹

date

通过date命令可以在命令行中查看系统的时间。

  • -d，按照给定的字符串显示日期，一般用于日期计算
  • 格式化字符串
    • %Y，年
    • %y，年份后两位数字（00-99）
    • %m，月份（01-12）
    • %d，日（01-31）
    • %H，小时（00-23）
    • %M，分钟（00-59）
    • %S，秒（00-60）
    • %s，自1970-01-01 00:00:00 UTC
      到现在的秒数

可以进行日期的加减：

Linux系统默认的时区可能并非中国的东八区时区。

  • 可以使用root权限修改时区

ntp

可以通过ntp程序自动校准系统时间。

ntpd服务启动后会定期联网校准系统时间。

也可以手动校准：ntpdate -u ntp.aliyun.com

压缩解压缩命令

Linux常用压缩格式：

  • tar格式，归档文件，简单讲文件整合到一个文件内，无压缩效果
  • gzip格式，gzip文件，不仅能整合到一个文件，同时有压缩体积的效果

tar

  • -c，创建压缩文件
  • -v，查看压缩、解压缩过程
  • -x，解压模式
  • -f，指定压缩、解压缩文件
  • -z，gzip模式
  • -C，指定解压路径

-z在开头使用，-f在结尾使用，-C单独使用

zip

  • -r，压缩文件夹

unzip

  • -d，指定要压缩的目录

scp

scp是cp命令的升级版，即ssh cp。

通过SSH协议完成不同Linux服务器之间文件的传输。

  • -r用于传输文件夹
  • 参数1，表示本机路径或远程目标路径
  • 参数2，表示远程目标路径或本机路径

Linux常用快捷键

常用快捷键

ctrl+C

  • 强制停止命令执行
  • 命令输入错误时，退出当前输入

ctrl+D

  • 退出账户登录
  • 退出特定程序（例如Python的交互环境）
  • 不能用于退出Vim/Vi

history

  • 查看历史输入过的命令

!+前缀字符

  • 自动匹配上一个前缀相同的命令

ctrl+R，输入命令

  • 匹配历史命令
  • 回车键直接执行
  • 键盘左右键，得到此命令（不执行）

ctrl+a，光标跳到命令开头

ctrl+e，光标跳到命令结尾

ctrl+左键，光标左跳一个单词

ctrl+右键，光标右跳一个单词

ctrl+L或者clear

  • 清空终端内容

04-Linux软件安装【Linux再学|黑马】

软件安装

操作系统安装软件的方式：

  • 下载安装包自行安装：
    • Windows系统，.exe文件、.msi文件等
    • Mac系统，.dmg文件、.pkg文件等
  • 系统的应用商店安装：
    • Windows系统，Microsoft Store
    • Mac系统，AppStore

CentOS，使用yum命令安装软件。

Ubuntu，使用apt命令安装软件。

yum

yum，CentOS的RPM包管理器，用于自动化安装配置Linux软件，并可以自动解决依赖问题。

  • -y，自动确认安装或卸载

yum命令需要root权限，需要联网。

apt

Ubuntu使用apt包管理器（安装.deb文件）。

用法和yum一致，也需要root权限和联网。

服务控制

Linux系统很多软件支持使用功能systemctl命令控制：启动、停止、开机自启。

能够被systemctl管理的软件，一般被称为服务

系统内置的服务比较多，如：

  • NetworkManager，主网络服务
  • network，副网络服务
  • firewalld，防火墙服务
  • sshd，ssh服务

第三方软件，只要有注册服务功能的，就可以通过systemctl进行控制。（也可以通过手动添加注册服务）

03 Linux权限控制【Linux再学|黑马】

超级管理员

Windows、MacOS、Linux均采用多用户的管理模式进行权限管理。

一般，普通用户在许多地方的操作权限都是受限的。

  • 例如，不能在根目录/下创建文件夹，等。
  • 普通用户的权限，一般在其HOME目录下是不受限的。在其HOME目录外一般仅有只读和执行权限，无修改权限。

Linux系统中，拥有最大权限的账户为root（超级管理员/根用户）。

su

用于切换账户的系统命令。

su：switch user

  • -符号，可选，表示是否在切换用户后加载环境变量（一般需要）。
  • 用户名，表示切换的用户，省略则表示切换到root用户。

切换用户后，可以使用exit命令（或者快捷键ctrl+D）退回到上一个用户。

普通用户切换需要输入密码，root用户切换不需要密码。

sudo

已知root用户密码时，可以通过su命令切换到root获取最大权限。

  • 但是，一般不建议长期使用root用户，避免带来系统损坏。

取而代之的是，使用sudo命令，给普通命令授权，临时以root身份执行该命令。

使用sudo命令，需要给普通用户配置sudo认证。

用户和用户组

Linux系统允许：

  • 配置多个用户
  • 配置多个用户组
  • 用户可以加入多个用户组

这意味着，Linux关于权限的管控级别有2个：

  • 针对用户的权限控制
  • 针对用户组的权限控制

用户组管理

以下命令需要root用户权限才能执行：

  • 创建用户组：groupadd 用户组名
  • 删除用户组：groupdel 用户组名

用户管理

以下命令需要root用户权限才能执行：

  • 创建用户：useradd [-g -d] 用户名
    • -g，指定用户的组。如果不指定，则创建同名组；如果所需的组已经存在，则必须使用-g。
    • -d，指定用户HOME路径。不指定，则默认在/home/<username>
  • 删除用户：userdel [-r] 用户名
    • -r，删除用户的HOME目录。不使用，则保留其HOME目录。
  • 查看用户所属组：id [用户名]
    • 用户名，参数，表示被查看的用户。不提供，则查看自身。
  • 修改用户所属组：usermod -aG 用户组 用户名
    • 将指定用户加入指定组

权限控制

权限信息表示

ls -l可以以列表形式查看目录内容和权限信息

10位信息，第一位为文件类型（文件夹、文件、软链接），后面3位一组分别表示用户、用户组、其他用户的权限。

rwx，分别表示读权限、写权限、执行权限。

chmod

修改文件、文件夹的权限信息。

只有文件、文件夹的所属用户、root用户才可以修改。

-R，表示对文件夹内所有内容应用相同操作。

例如，chmod u=rwx,g=rx,o=x hello.txt，表示将文件权限修改为rwxr-x--x。

权限的数字表示：r记为4，w记为2，x记为1。

  • 则rwxr-x--x可以表示为751

chown

修改文件、文件夹的所属用户和用户组。

普通用户不能修改所属用户或用户组，只有root用户才适用。

-R，表示对文件夹内所有内容应用相同操作。

例如，chown root:alluser a.txt，将文件所属用户改为root，用户组改为alluser。
