Linux常用快捷键

常用快捷键

ctrl+C

  • 强制停止命令执行
  • 命令输入错误时，退出当前输入

ctrl+D

  • 退出账户登录
  • 退出特定程序（例如Python的交互环境）
  • 不能用于退出Vim/Vi

history

  • 查看历史输入过的命令

!+前缀字符

  • 自动匹配上一个前缀相同的命令

ctrl+R，输入命令

  • 匹配历史命令
  • 回车键直接执行
  • 键盘左右键，得到此命令（不执行）

ctrl+a，光标跳到命令开头

ctrl+e，光标跳到命令结尾

ctrl+左键，光标左跳一个单词

ctrl+右键，光标右跳一个单词

ctrl+L或者clear

  • 清空终端内容

04-Linux软件安装【Linux再学|黑马】

软件安装

操作系统安装软件的方式：

  • 下载安装包自行安装：
    • Windows系统，.exe文件、.msi文件等
    • Mac系统，.dmg文件、.pkg文件等
  • 系统的应用商店安装：
    • Windows系统，Microsoft Store
    • Mac系统，AppStore

CentOS，使用yum命令安装软件。

Ubuntu，使用apt命令安装软件。

yum

yum，CentOS的RPM包管理器，用于自动化安装配置Linux软件，并可以自动解决依赖问题。

  • -y，自动确认安装或卸载

yum命令需要root权限，需要联网。

apt

Ubuntu使用apt包管理器（安装.deb文件）。

用法和yum一致，也需要root权限和联网。

服务控制

Linux系统很多软件支持使用功能systemctl命令控制：启动、停止、开机自启。

能够被systemctl管理的软件，一般被称为服务

系统内置的服务比较多，如：

  • NetworkManager，主网络服务
  • network，副网络服务
  • firewalld，防火墙服务
  • sshd，ssh服务

第三方软件，只要有注册服务功能的，就可以通过systemctl进行控制。（也可以通过手动添加注册服务）

03 Linux权限控制【Linux再学|黑马】

超级管理员

Windows、MacOS、Linux均采用多用户的管理模式进行权限管理。

一般，普通用户在许多地方的操作权限都是受限的。

  • 例如，不能在根目录/下创建文件夹，等。
  • 普通用户的权限，一般在其HOME目录下是不受限的。在其HOME目录外一般仅有只读和执行权限，无修改权限。

Linux系统中，拥有最大权限的账户为root（超级管理员/根用户）。

su

用于切换账户的系统命令。

su：switch user

  • -符号，可选，表示是否在切换用户后加载环境变量（一般需要）。
  • 用户名，表示切换的用户，省略则表示切换到root用户。

切换用户后，可以使用exit命令（或者快捷键ctrl+D）退回到上一个用户。

普通用户切换需要输入密码，root用户切换不需要密码。

sudo

已知root用户密码时，可以通过su命令切换到root获取最大权限。

  • 但是，一般不建议长期使用root用户，避免带来系统损坏。

取而代之的是，使用sudo命令，给普通命令授权，临时以root身份执行该命令。

使用sudo命令，需要给普通用户配置sudo认证。

为普通用户配置sudo认证

用户和用户组

Linux系统允许：

  • 配置多个用户
  • 配置多个用户组
  • 用户可以加入多个用户组

这意味着，Linux关于权限的管控级别有2个：

  • 针对用户的权限控制
  • 针对用户组的权限控制

用户组管理

以下命令需要root用户权限才能执行：

  • 创建用户组：groupadd 用户组名
  • 删除用户组：groupdel 用户组名

用户管理

以下命令需要root用户权限才能执行：

  • 创建用户：useradd [-g -d] 用户名
    • -g，指定用户的组。如果不指定，则创建同名组；如果所需的组已经存在，则必须使用-g。
    • -d，指定用户HOME路径。不指定，则默认在/home/<username>
  • 删除用户：userdel [-r] 用户名
    • -r，删除用户的HOME目录。不使用，则保留其HOME目录。
  • 查看用户所属组：id [用户名]
    • 用户名，参数，表示被查看的用户。不提供，则查看自身。
  • 修改用户所属组：usermod -aG 用户组 用户名
    • 将指定用户加入指定组

权限控制

权限信息表示

ls -l可以以列表形式查看目录内容和权限信息

10位信息，第一位为文件类型（文件夹、文件、软链接），后面3位一组分别表示用户、用户组、其他用户的权限。

rwx，分别表示读权限、写权限、执行权限。

chmod

修改文件、文件夹的权限信息。

只有文件、文件夹的所属用户、root用户才可以修改。

-R，表示对文件夹内所有内容应用相同操作。

例如，chmod u=rwx,g=rx,o=x hello.txt，表示将文件权限修改为rwxr-x--x。

权限的数字表示：r记为4，w记为2，x记为1。

  • 则rwxr-x--x可以表示为751

chown

修改文件、文件夹的所属用户和用户组。

普通用户不能修改所属用户或用户组，只有root用户才适用。

-R，表示对文件夹内所有内容应用相同操作。

例如，chown root:alluser a.txt，将文件所属用户改为root，用户组改为alluser。

02 Linux基本命令【Linux再学|黑马】

路径

相对路径：以根目录为起点，描述路径的方式，路径以/开头。

相对路径：以当前目录为起点，描述路径的方式，路径不需要/开头。

特殊路径符：

  • 当前目录：.。
    • 如，cd .或cd ./Desktop
  • 上一级目录：..。
    • 如，cd ..或cd ../..
  • 用户的HOME目录：~。
    • 如，cd ~或cd ~/Desktop

Linux文件相关命令

学习Linux，本质是学习在命令行下熟练使用Linux的各类命令。

命令行：Linux终端（Terminal），是一种命令提示符页面。以纯字符的形式操作系统，可以使用各种字符画命令对系统发出操作指令。

命令：Linux程序。命令没有图形化界面，可以在命令行（终端）中提供字符化的反馈。

  • 命令的本体是二进制可执行程序，和Windows系统中的.exe文件类似；执行一个命令，实质上是运行一个二进制可执行程序。

命令的通用格式：command [-options] [parameter]

ls

命令行中平铺展示当前目录（默认为HOME目录）下的文件或文件夹。

  • HOME目录：每个用户在Linux系统的专属目录（以~表示），普通用户默认为/home/<username>，超级用户为/root
  • 当前工作目录：Linux命令行在执行命令的时候，需要一个工作目录。命令行默认设置的工作目录是用户的HOME目录。
  • -a：列出全部文件（包括隐藏文件/文件夹，以.开头的文件）
  • -l：以列表（竖向排列）形式展示文件/文件夹
  • -h：和-l选项合用，合理显示文件大小单位

命令选项可以组合使用，如：ls -alh

cd

更改当前目录。

cd：change directory

无选项。

不写参数，则回到用户的HOME目录；写参数，则转到对应的目录下。

pwd

查看当前工作目录

pwd：print work directory

无选项，无参数。

mkdir

创建新的目录（文件夹）

mkdir：make directory

参数必填（即要创建的文件夹路径），可以是相对路径或绝对路径

-p表示自动创建不存在的父目录，适用于创建连续多层级的目录。

touch

创建文件

无选项，参数必填，表示要创建的文件路径。

cat

查看文件内容。

无选项，参数必填。

more

cat直接将文件内容全部显示出来。

more支持翻页查看，文件内容过多时，可以一页一页展示。

无选项，参数必填。

查看过程中，使用空格键翻页；通过q退出查看。

cp

复制文件或文件夹.

cp：copy

  • -r，可选，用于递归复制文件夹
  • 参数1，表示被复制的路径
  • 参数2，表示复制到的路径

mv

移动文件或文件夹。

mv：move

  • 参数1，表示被移动的路径
  • 参数2，表示移动到的路径；如果目标不存在，则进行改名

rm

移除文件夹。

rm：remove

  • -r，表示递归删除文件夹
  • -f，表示强制删除（不提示，一般用于root用户）
  • 参数，为被删除文件路径，支持多个（空格相隔），支持通配符*做模糊匹配

Linux查找命令

which

通过which命令，可以查看其他命令的程序文件存放路径。

find

通过find命令可以搜索指定的文件。

以文件名搜索

支持使用通配符*做模糊查询：

  • test*，表示以test开头的内容
  • *test，表示以test结尾的内容
  • *test*，表示包含test的内容

以文件大小搜索

  • +、-表示大于和小于
  • n表示大小数字
  • kMG表示文件大小单位分别为kB、MB、GB

例如，find / -size +100M，为搜索根路径下大小在100MB以上的文件。

Linux管道命令

grep

从文件中通过关键字过滤文件行。

  • -n，可选，表示在结果中显示匹配行的行号。
  • 关键字，必填，表示过滤的字符，带有空格或其他特殊符号需要用" "将关键字整体引起来。
  • 文件路径，必填，表示被过滤文件路径，可以作为内容输入端口。

wc

统计文件的行数、字符数等信息。

  • -c，统计字节数
  • -m，统计字符数
  • -l，统计行数
  • -w，统计单词数
  • 文件路径，必填，表示被统计的文件，可以作为内容输入端口
