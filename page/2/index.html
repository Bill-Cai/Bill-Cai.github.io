06-Linux进程管理

进程

程序运行在操作系统中时，是被操作系统所管理的。

为管理运行的程序，每个程序在运行时都会在操作系统中注册为系统中的一个进程。

并且，操作系统会给每个进程分配一个独有的进程号（进程ID）。

查看进程

通过ps命令查看Linux系统中的进程信息。

  • -e，显示出全部的进程
  • -f，以完全格式化的形式展示信息（展示全部信息）

一般，固定使用ps -ef列出全部进程的全部信息。

关闭进程

在Linux中，通过kill命令关闭进程。

  • -9，表示强制关闭进程。如果不使用该选项，则会向该进程发送关闭信息，能否成功关闭需要看进程自身的处理机制。

主机状态

可以通过top命令查看CPU、内存使用情况，类似于Windows的任务管理器。

  • 默认每5秒刷新一次。
  • 通过q或ctrl+C退出。

top命令支持选项：

  • -p，只显示某个进程信息
  • -d，设置刷新时间，默认5秒
  • -c，显示产生进程的完整命令，默认是进程名
  • -n，指定刷新次数
  • -b，以批次的非交互非全屏模式运行。一般配合-n指定输出次数并将输出重定向到指定文件，如top -b -n 3 > /tmp/top.info
  • -i，不显示任何闲置（idle）或无用（zombie）进程
  • -u，查找特定用户启动的进程

top命令交互运行状态下，可以使用H（显示帮助画面）、C（显示完整命令）等按键实现功能。

磁盘信息

通过df命令查看硬盘的使用情况。

通过iostat查看CPU、磁盘的相关信息。

  • -x，显示更多信息
  • 参数num1表示刷新间隔，参数num2表示刷新次数

网络状态

通过sar查看网络的相关信息。

  • -n，查看网络，DEV表示网络接口
  • 参数num1表示刷新间隔，参数num2表示刷新次数

05-Linux网络

网络传输

ping

检查指定的网络服务器是否为可联通状态。

  • -c，表示检查次数，不使用则会无限次持续检查

wget

非交互式的文件下载器，可以在命令行内下载网络文件。

  • -b，可行，表示后台下载，会将日志写入当前工作目录的wget-log文件。

后台下载，可以通过tail -f wget-log实时监控下载进度。

无论下载是否完成，都会生成下载的文件。

curl

发生http请求，可用于：下载文件、获取信息等。

  • -O，表示在链接为下载链接时保存文件。

端口

端口，设备和外界通讯交流的出入口，分为：

  • 物理端口：接口，如USB接口、HDMI端口等
  • 虚拟端口：计算机内部的端口，不可见，用于操作系统和外部交互。

计算机程序之间的通讯，使用IP来锁定计算机，使用端口来锁定计算机上的具体的程序。

Linux系统支持65535个端口，分3类使用：

  • 公认端口：1~1023，通常用于系统内置或知名程序的预留使用，如SSH服务的22端口，HTTPS服务的443端口等。（一般不能占用这个敢为的端口）
  • 注册端口：1024~49151，通常可以随意使用，用于松散绑定一些程序或服务。
  • 动态端口：49152~65535，同城不会固定绑定程序，而是当程序对外进行网络链接时临时使用。

Linux IP和主机名

IP地址

联网电脑都会有一个IP地址，用于和其他计算机进行通讯。

IP地址的两个版本：IPv4，IPv6。

IPv4的格式：xxx.xxx.xxx.xxx（xxx表示0~255的数字）

通过ifconfig可以查看本机的IP地址

  • 可以通过yum -y install net-tools进行安装

特殊IP地址

127.0.0.1，用于指代本机

0.0.0.0，用于指代本机，或在端口绑定中确定绑定关系，或在IP地址限制中表示允许任意IP访问。

主机名

每台电脑，除了IP地址外，还有一个主机名。

查看主机名：hostname

主机名可以进行设置。

修改主机名（需要root权限）：hostnamectl set-hostname 主机名

  • 重新登录终端可以看到主机名修改成功。

域名解析

IP地址难以记忆，可以通过字符化的地址去访问服务器，称之为域名。域名和IP之间的映射关系，通过域名解析实现。

以百度为例分析其访问流程：

  • 访问www.baidu.com
    • 检查本地是否有www.baidu.com的IP地址（Windows系统的C:\Windows\System32\drivers\etc\hosts文件；Linux系统的/etc/hosts文件）
      • 有，则直接访问IP地址，打开网站
      • 否则，联网查询公开DNS服务器（如114.114.114.114，8.8.8.8等），查询是否有www.baidu.com的IP地址，获取后再访问

Linux其他命令

软链接

ln命令在系统中创建软链接（标记为l），可以将文件、文件夹链接到其他位置。

  • 参数1，表示被连链接的原文件或文件夹
  • 参数2，表示要链接去的文件或文件夹

date

通过date命令可以在命令行中查看系统的时间。

  • -d，按照给定的字符串显示日期，一般用于日期计算
  • 格式化字符串
    • %Y，年
    • %y，年份后两位数字（00-99）
    • %m，月份（01-12）
    • %d，日（01-31）
    • %H，小时（00-23）
    • %M，分钟（00-59）
    • %S，秒（00-60）
    • %s，自1970-01-01 00:00:00 UTC
      到现在的秒数

Linux系统默认的时区可能并非中国的东八区时区。

  • 可以使用root权限修改时区

ntp

可以通过ntp程序自动校准系统时间。

ntpd服务启动后会定期联网校准系统时间。

也可以手动校准：ntpdate -u ntp.aliyun.com

压缩解压缩命令

Linux常用压缩格式：

  • tar格式，归档文件，简单讲文件整合到一个文件内，无压缩效果
  • gzip格式，gzip文件，不仅能整合到一个文件，同时有压缩体积的效果

tar

  • -c，创建压缩文件
  • -v，查看压缩、解压缩过程
  • -x，解压模式
  • -f，指定压缩、解压缩文件
  • -z，gzip模式
  • -C，指定解压路径

-z在开头使用，-f在结尾使用，-C单独使用

zip

  • -r，压缩文件夹

unzip

  • -d，指定要压缩的目录

scp

scp是cp命令的升级版，即ssh cp。

通过SSH协议完成不同Linux服务器之间文件的传输。

  • -r用于传输文件夹
  • 参数1，表示本机路径或远程目标路径
  • 参数2，表示远程目标路径或本机路径
