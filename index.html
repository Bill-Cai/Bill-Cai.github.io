02 Linux基本命令【Linux再学|黑马】

路径

相对路径：以根目录为起点，描述路径的方式，路径以/开头。

相对路径：以当前目录为起点，描述路径的方式，路径不需要/开头。

特殊路径符：

  • 当前目录：.。
    • 如，cd .或cd ./Desktop
  • 上一级目录：..。
    • 如，cd ..或cd ../..
  • 用户的HOME目录：~。
    • 如，cd ~或cd ~/Desktop

Linux文件相关命令

学习Linux，本质是学习在命令行下熟练使用Linux的各类命令。

命令行：Linux终端（Terminal），是一种命令提示符页面。以纯字符的形式操作系统，可以使用各种字符画命令对系统发出操作指令。

命令：Linux程序。命令没有图形化界面，可以在命令行（终端）中提供字符化的反馈。

  • 命令的本体是二进制可执行程序，和Windows系统中的.exe文件类似；执行一个命令，实质上是运行一个二进制可执行程序。

命令的通用格式：command [-options] [parameter]

ls

命令行中平铺展示当前目录（默认为HOME目录）下的文件或文件夹。

  • HOME目录：每个用户在Linux系统的专属目录（以~表示），普通用户默认为/home/<username>，超级用户为/root
  • 当前工作目录：Linux命令行在执行命令的时候，需要一个工作目录。命令行默认设置的工作目录是用户的HOME目录。
  • -a：列出全部文件（包括隐藏文件/文件夹，以.开头的文件）
  • -l：以列表（竖向排列）形式展示文件/文件夹
  • -h：和-l选项合用，合理显示文件大小单位

命令选项可以组合使用，如：ls -alh

cd

更改当前目录。

cd：change directory

无选项。

不写参数，则回到用户的HOME目录；写参数，则转到对应的目录下。

pwd

查看当前工作目录

pwd：print work directory

无选项，无参数。

mkdir

创建新的目录（文件夹）

mkdir：make directory

参数必填（即要创建的文件夹路径），可以是相对路径或绝对路径

-p表示自动创建不存在的父目录，适用于创建连续多层级的目录。

touch

创建文件

无选项，参数必填，表示要创建的文件路径。

cat

查看文件内容。

无选项，参数必填。

more

cat直接将文件内容全部显示出来。

more支持翻页查看，文件内容过多时，可以一页一页展示。

无选项，参数必填。

查看过程中，使用空格键翻页；通过q退出查看。

cp

复制文件或文件夹.

cp：copy

  • -r，可选，用于递归复制文件夹
  • 参数1，表示被复制的路径
  • 参数2，表示复制到的路径

mv

移动文件或文件夹。

mv：move

  • 参数1，表示被移动的路径
  • 参数2，表示移动到的路径；如果目标不存在，则进行改名

rm

移除文件夹。

rm：remove

  • -r，表示递归删除文件夹
  • -f，表示强制删除（不提示，一般用于root用户）
  • 参数，为被删除文件路径，支持多个（空格相隔），支持通配符*做模糊匹配

Linux查找命令

which

通过which命令，可以查看其他命令的程序文件存放路径。

find

通过find命令可以搜索指定的文件。

以文件名搜索

支持使用通配符*做模糊查询：

  • test*，表示以test开头的内容
  • *test，表示以test结尾的内容
  • *test*，表示包含test的内容

以文件大小搜索

  • +、-表示大于和小于
  • n表示大小数字
  • kMG表示文件大小单位分别为kB、MB、GB

例如，find / -size +100M，为搜索根路径下大小在100MB以上的文件。

Linux管道命令

grep

从文件中通过关键字过滤文件行。

  • -n，可选，表示在结果中显示匹配行的行号。
  • 关键字，必填，表示过滤的字符，带有空格或其他特殊符号需要用" "将关键字整体引起来。
  • 文件路径，必填，表示被过滤文件路径，可以作为内容输入端口。

wc

统计文件的行数、字符数等信息。

  • -c，统计字节数
  • -m，统计字符数
  • -l，统计行数
  • -w，统计单词数
  • 文件路径，必填，表示被统计的文件，可以作为内容输入端口

Windows上的Linux子系统：WSL

WSL

WSL（Windows Subsystem for Linux），是用于Windows系统之上的Linux子系统。可以在Windows系统中获得Linux系统环境，并完全直连计算机硬件，无需通过虚拟机虚拟硬件。

Win10开启WSL：设置->程序和功能->启动或关闭Windows功能->勾选“适用于Linux的Windows子系统”

Win10的Microsoft Store中可以下载安装Ubuntu和Microsoft Terminal。

WSL Ubuntu

WSL Ubuntu默认普通用户，第一次使用时，需要设置用户名和密码。

如果需要登录root用户，可以：

  1. 在PowerShell中修改默认登录用户为root：ubuntu config --default-user root。
  2. 以root进入Ubuntu终端后使用passwd更改密码。
  3. 再将默认登录用户改为普通用户。
  4. 在Ubuntu终端使用su命令，并输入密码，激活root用户。
  5. 通过exit命令退出到普通用户。

需要注意，root用户有最高权限，需要慎重使用，避免错误操作对系统造成损坏。

01 Linux基础【Linux再学|黑马】

前置知识

计算机组成：硬件，软件。

操作系统是软件的一类，协助用户调度硬件工作。

常见操作系统：

  • PC端：Windows、Linux、MacOS
  • 移动端：Android、IOS、HarmonyOS

Linux组成

Linux系统组成：Linux系统内核+系统级应用程序

  • 内核，提供系统最核心的功能：调度硬件工作。
  • 系统级应用程序，出厂自带程序，供用户快速上手操作系统。

Linux内核

Linux内核免费开源：The Linux Kernel Archives

Linux发行版

Linux内核的免费开源，使得任何人都可以获得并修改内核，自行集成系统级程序。

提供了内核+系统级程序的完整封装，称为Linux发行版。

比较常用的：CentOS，Ubuntu等。

虚拟机

通过虚拟化技术，在电脑内虚拟出计算机硬件，并给虚拟的硬件安装操作系统，即可获得一台虚拟的电脑，称为虚拟机。

可以用VMware Workstation虚拟机软件。（这破玩意儿太重了，不喜欢）

使用Linux系统，多数用的是命令行，而不是图形化。

  • Linux图形化优化不是重点，图形化页面不好用不稳定。
  • 命令行效率更高、更直观、资源占用低、程序运行更稳定。

可以用XShell、MobaXterm等软件远程连接到Linux系统。

  • 使用ipconfig查看IP地址。

虚拟机快照，可以保存虚拟机的状态，当虚拟机出现问题时候可以通过预先制作的快照恢复到制作时候的状态，作备份用。

Linux目录结构

Windows系统可以有多个盘符；Linux只有一个根目录/。

  • Windows系统中，路径之间的层级关系，使用\表示，如D:\data\a.txt。
  • Linux系统中，路径之间的层级关系，使用/表示，如/home/Amy/b.txt。

DBSCAN - 原理和实现

概念

DBSCAN（Density-Based Spatial Clustering of Applications with Noise），有噪声的基于密度聚类算法。

  • 将簇定义为具有足够高密度的区域；
  • 可以在有噪声的空间数据中发现任意形状的聚类。

关于DBSCAN的一些定义：

  • $E$邻域：对于给定对象，半径为Ε内的区域。
  • 核心对象：$E$邻域内样本点数大于等于$MinPts$的给定对象。
  • 直接密度可达：对于样本集合$D$，如果样本点$q$在$p$的$Ε$邻域内，并且$p$为核心对象，那么对象$q$从对象$p$直接密度可达。
  • 密度可达：对于样本集合$D$，给定一串样本点$p_1,p_2,…,p_n$，$p=p_1$，$q=p_n$，假如对象$p_i$从$p_{i-1}$直接密度可达，那么对象$q$从对象$p$密度可达。
  • 密度相连：存在样本集合$D$中的一点$o$，如果对象$o$到对象$p$和对象$q$都是密度可达的，那么$p$和$q$密度相联。

DBSCAN目的是找到密度相连对象的最大集合。
