05-Linux网络

网络传输

ping

检查指定的网络服务器是否为可联通状态。

  • -c，表示检查次数，不使用则会无限次持续检查

wget

非交互式的文件下载器，可以在命令行内下载网络文件。

  • -b，可行，表示后台下载，会将日志写入当前工作目录的wget-log文件。

后台下载，可以通过tail -f wget-log实时监控下载进度。

无论下载是否完成，都会生成下载的文件。

curl

发生http请求，可用于：下载文件、获取信息等。

  • -O，表示在链接为下载链接时保存文件。

端口

端口，设备和外界通讯交流的出入口，分为：

  • 物理端口：接口，如USB接口、HDMI端口等
  • 虚拟端口：计算机内部的端口，不可见，用于操作系统和外部交互。

计算机程序之间的通讯，使用IP来锁定计算机，使用端口来锁定计算机上的具体的程序。

Linux系统支持65535个端口，分3类使用：

  • 公认端口：1~1023，通常用于系统内置或知名程序的预留使用，如SSH服务的22端口，HTTPS服务的443端口等。（一般不能占用这个敢为的端口）
  • 注册端口：1024~49151，通常可以随意使用，用于松散绑定一些程序或服务。
  • 动态端口：49152~65535，同城不会固定绑定程序，而是当程序对外进行网络链接时临时使用。

Linux IP和主机名

IP地址

联网电脑都会有一个IP地址，用于和其他计算机进行通讯。

IP地址的两个版本：IPv4，IPv6。

IPv4的格式：xxx.xxx.xxx.xxx（xxx表示0~255的数字）

通过ifconfig可以查看本机的IP地址

  • 可以通过yum -y install net-tools进行安装

特殊IP地址

127.0.0.1，用于指代本机

0.0.0.0，用于指代本机，或在端口绑定中确定绑定关系，或在IP地址限制中表示允许任意IP访问。

主机名

每台电脑，除了IP地址外，还有一个主机名。

查看主机名：hostname

主机名可以进行设置。

修改主机名（需要root权限）：hostnamectl set-hostname 主机名

  • 重新登录终端可以看到主机名修改成功。

域名解析

IP地址难以记忆，可以通过字符化的地址去访问服务器，称之为域名。域名和IP之间的映射关系，通过域名解析实现。

以百度为例分析其访问流程：

  • 访问www.baidu.com
    • 检查本地是否有www.baidu.com的IP地址（Windows系统的C:\Windows\System32\drivers\etc\hosts文件；Linux系统的/etc/hosts文件）
      • 有，则直接访问IP地址，打开网站
      • 否则，联网查询公开DNS服务器（如114.114.114.114，8.8.8.8等），查询是否有www.baidu.com的IP地址，获取后再访问

Linux其他命令

软链接

ln命令在系统中创建软链接（标记为l），可以将文件、文件夹链接到其他位置。

  • 参数1，表示被连链接的原文件或文件夹
  • 参数2，表示要链接去的文件或文件夹

date

通过date命令可以在命令行中查看系统的时间。

  • -d，按照给定的字符串显示日期，一般用于日期计算
  • 格式化字符串
    • %Y，年
    • %y，年份后两位数字（00-99）
    • %m，月份（01-12）
    • %d，日（01-31）
    • %H，小时（00-23）
    • %M，分钟（00-59）
    • %S，秒（00-60）
    • %s，自1970-01-01 00:00:00 UTC
      到现在的秒数

Linux系统默认的时区可能并非中国的东八区时区。

  • 可以使用root权限修改时区

ntp

可以通过ntp程序自动校准系统时间。

ntpd服务启动后会定期联网校准系统时间。

也可以手动校准：ntpdate -u ntp.aliyun.com

Linux常用快捷键

常用快捷键

ctrl+C

  • 强制停止命令执行
  • 命令输入错误时，退出当前输入

ctrl+D

  • 退出账户登录
  • 退出特定程序（例如Python的交互环境）
  • 不能用于退出Vim/Vi

history

  • 查看历史输入过的命令

!+前缀字符

  • 自动匹配上一个前缀相同的命令

ctrl+R，输入命令

  • 匹配历史命令
  • 回车键直接执行
  • 键盘左右键，得到此命令（不执行）

ctrl+a，光标跳到命令开头

ctrl+e，光标跳到命令结尾

ctrl+左键，光标左跳一个单词

ctrl+右键，光标右跳一个单词

ctrl+L或者clear

  • 清空终端内容

04-Linux软件安装【Linux再学|黑马】

软件安装

操作系统安装软件的方式：

  • 下载安装包自行安装：
    • Windows系统，.exe文件、.msi文件等
    • Mac系统，.dmg文件、.pkg文件等
  • 系统的应用商店安装：
    • Windows系统，Microsoft Store
    • Mac系统，AppStore

CentOS，使用yum命令安装软件。

Ubuntu，使用apt命令安装软件。

yum

yum，CentOS的RPM包管理器，用于自动化安装配置Linux软件，并可以自动解决依赖问题。

  • -y，自动确认安装或卸载

yum命令需要root权限，需要联网。

apt

Ubuntu使用apt包管理器（安装.deb文件）。

用法和yum一致，也需要root权限和联网。

服务控制

Linux系统很多软件支持使用功能systemctl命令控制：启动、停止、开机自启。

能够被systemctl管理的软件，一般被称为服务

系统内置的服务比较多，如：

  • NetworkManager，主网络服务
  • network，副网络服务
  • firewalld，防火墙服务
  • sshd，ssh服务

第三方软件，只要有注册服务功能的，就可以通过systemctl进行控制。（也可以通过手动添加注册服务）
