个人博客搭建 - 基于Hexo

搭建思路

  1. 安装Hexo
    1. 安装node.js环境（node，npm）
    2. 局部安装Hexo：npm install hexo
  2. 新建blog文件夹，并cd到blog文件夹下
  3. 通过npx hexo init初始化
  4. 本地启动hexo服务：npx hexo server或者简写为npx hexo s
  5. 访问默认的http://localhost:4000/网页查看

新建博客

  1. 创建md文件：npx hexo n "blog-file-name"
  2. 删除hexo生成的所有文档（其实就是清理public文件夹）：npx hexo clean
  3. 重新生成页面文档：npx hexo g
  4. 启动服务，访问博客网页

网页部署

  1. 注册GitHub账号，创建代码仓库，要求仓库名为<username>.github.io，因为GitHub Page解析的时候是找的这个名称

  2. 安装Hexo依赖：npm install --save hexo-deployer-git

  3. 修改配置文件_config.yml，在最下方的deploy部分添加git配置

  4. 部署（上传至GitHub）：npx hexo d
