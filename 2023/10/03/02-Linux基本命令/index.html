02 Linux基本命令【Linux再学|黑马】

路径

相对路径：以根目录为起点，描述路径的方式，路径以/开头。

相对路径：以当前目录为起点，描述路径的方式，路径不需要/开头。

特殊路径符：

  • 当前目录：.。
    • 如，cd .或cd ./Desktop
  • 上一级目录：..。
    • 如，cd ..或cd ../..
  • 用户的HOME目录：~。
    • 如，cd ~或cd ~/Desktop

Linux文件相关命令

学习Linux，本质是学习在命令行下熟练使用Linux的各类命令。

命令行：Linux终端（Terminal），是一种命令提示符页面。以纯字符的形式操作系统，可以使用各种字符画命令对系统发出操作指令。

命令：Linux程序。命令没有图形化界面，可以在命令行（终端）中提供字符化的反馈。

  • 命令的本体是二进制可执行程序，和Windows系统中的.exe文件类似；执行一个命令，实质上是运行一个二进制可执行程序。

命令的通用格式：command [-options] [parameter]

ls

命令行中平铺展示当前目录（默认为HOME目录）下的文件或文件夹。

  • HOME目录：每个用户在Linux系统的专属目录（以~表示），普通用户默认为/home/<username>，超级用户为/root
  • 当前工作目录：Linux命令行在执行命令的时候，需要一个工作目录。命令行默认设置的工作目录是用户的HOME目录。
  • -a：列出全部文件（包括隐藏文件/文件夹，以.开头的文件）
  • -l：以列表（竖向排列）形式展示文件/文件夹
  • -h：和-l选项合用，合理显示文件大小单位

命令选项可以组合使用，如：ls -alh

cd

更改当前目录。

cd：change directory

无选项。

不写参数，则回到用户的HOME目录；写参数，则转到对应的目录下。

pwd

查看当前工作目录

pwd：print work directory

无选项，无参数。

mkdir

创建新的目录（文件夹）

mkdir：make directory

参数必填（即要创建的文件夹路径），可以是相对路径或绝对路径

-p表示自动创建不存在的父目录，适用于创建连续多层级的目录。

touch

创建文件

无选项，参数必填，表示要创建的文件路径。

cat

查看文件内容。

无选项，参数必填。

more

cat直接将文件内容全部显示出来。

more支持翻页查看，文件内容过多时，可以一页一页展示。

无选项，参数必填。

查看过程中，使用空格键翻页；通过q退出查看。

cp

复制文件或文件夹.

cp：copy

  • -r，可选，用于递归复制文件夹
  • 参数1，表示被复制的路径
  • 参数2，表示复制到的路径

mv

移动文件或文件夹。

mv：move

  • 参数1，表示被移动的路径
  • 参数2，表示移动到的路径；如果目标不存在，则进行改名

rm

移除文件夹。

rm：remove

  • -r，表示递归删除文件夹
  • -f，表示强制删除（不提示，一般用于root用户）
  • 参数，为被删除文件路径，支持多个（空格相隔），支持通配符*做模糊匹配

Linux查找命令

which

通过which命令，可以查看其他命令的程序文件存放路径。

find

通过find命令可以搜索指定的文件。

以文件名搜索

支持使用通配符*做模糊查询：

  • test*，表示以test开头的内容
  • *test，表示以test结尾的内容
  • *test*，表示包含test的内容

以文件大小搜索

  • +、-表示大于和小于
  • n表示大小数字
  • kMG表示文件大小单位分别为kB、MB、GB

例如，find / -size +100M，为搜索根路径下大小在100MB以上的文件。

Linux管道命令

grep

从文件中通过关键字过滤文件行。

  • -n，可选，表示在结果中显示匹配行的行号。
  • 关键字，必填，表示过滤的字符，带有空格或其他特殊符号需要用" "将关键字整体引起来。
  • 文件路径，必填，表示被过滤文件路径，可以作为内容输入端口。

wc

统计文件的行数、字符数等信息。

  • -c，统计字节数
  • -m，统计字符数
  • -l，统计行数
  • -w，统计单词数
  • 文件路径，必填，表示被统计的文件，可以作为内容输入端口

tail

查看文件尾部内容，跟踪文件的最新更改。

  • -f，表示持续跟踪；ctrl+C退出
  • -num，表示查看尾部行数，默认为10行（相当于-10）
  • 参数，必填，表示被跟踪的文件路径
